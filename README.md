## ✨ Features

*   **🎶 High-Quality Conversion**: Convert MP3 to **WAV (44.1kHz/24-bit & 48kHz/32-bit)** and **AIFF (44.1kHz/24-bit)** for professional use.
*   **📁 Batch Processing**: Convert multiple MP3 files at once to save time.
*   **🖥️ Clean, Intuitive GUI**: A user-friendly interface built with Python's Tkinter, making it accessible for everyone.
*   **⚡ Fast & Reliable**: Uses FFmpeg for robust and efficient audio processing.
*   **🐍 Open Source**: Built with Python, inviting contributions and customization.

## Quick Start

### Prerequisites

Before you begin, ensure you have the following installed on your system:

*   **FFmpeg**: This is essential for the app to work.
    *   **macOS**: `brew install ffmpeg`
    *   **Windows**: Download from [ffmpeg.org](https://ffmpeg.org/download.html) and add to your system's PATH.
    *   **Linux (Ubuntu/Debian)**: `sudo apt install ffmpeg`
